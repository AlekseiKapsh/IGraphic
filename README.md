Интерфейс IGraphic — общий контракт с методом Draw()
Три листовых класса (Line, Rectangle, Circle) — каждый хранит свои параметры и умеет себя "рисовать" (выводить информацию в консоль)
Класс CompositeGraphic — контейнер, хранит список дочерних элементов, при вызове Draw() рекурсивно обходит всех потомков
Вызов rootGroup.Draw() рекурсивно проходит всю структуру и выводит каждый элемент с отступами, показывая вложенность
